StaticPageGenerator.php : 19

Generates static HTML snapshots of public pages (posts/ and events/) and saves them under /pregenerated so that index.php can serve them directly without going through PHP routing for each anonymous request.

File layout inside /pregenerated: posts/index_1.html → /posts or /posts?page=1 posts/index_2.html → /posts?page=2 posts/5.html → /posts/5 events/index_1.html → /events or /events?page=1 events/5.html → /events/5
