query_builder.php : 12

Chainable SQL query builder for ActiveModel records and relation scopes.

Usage: Post::where(['author_id' => 1])->where(['status' => 'published'])->get(); Post::where(['author_id' => 1])->first(); Post::where(['author_id' => 1])->count(); Post::where('author_id', 1)->orderBy('created_at', 'DESC')->limit(10)->get();
